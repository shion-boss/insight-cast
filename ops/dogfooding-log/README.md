# ドッグフーディング使用記録

Insight Cast を自社で使ったときの記録。「使った人目線」での気づき・違和感・良かった点を残す。

## ファイル命名規則

```
YYYY-MM-DD-<内容の一言>.md
```

例: `2026-04-25-mint-interview-first.md`

## 記録テンプレート

```markdown
# YYYY-MM-DD ドッグフーディング記録

## 実施内容
- 使ったキャスト:
- 取材テーマ:
- インタビュー所要時間:

## 良かった点

## 気になった点（UX・コピー・AIキャストの振る舞い等）

## 修正を依頼したい点
```

# Blog Theme Analysis

## 使う場面

- 既存ブログの偏りを見たい時
- インタビュー前に「まだ書かれていない価値」を探したい時
- `growth` や `arch` が次のテーマ候補を絞りたい時

## 読むファイル

- `README.md`
- `prompt.md`

## 期待する出力

- 頻出テーマ
- 空白テーマ
- インタビューで優先して聞くべきこと
- だれのキャストが向いているか

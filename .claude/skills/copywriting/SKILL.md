# Copywriting Core Skill

## このスキルの位置づけ

`.claude/skills/copywriting-lp/SKILL.md` がフレームワーク・構成・チェックリストを扱うのに対して、
このスキルは **「どう組み立てるか」という思考法** を扱う。

具体的には以下を対象とする。

- コピーの**論証構造**（A否定→B提示、対比、AでもBでもなくC の使い分け）
- 文章の**リズムと呼応**（opening と closing の構文を対応させる等）
- 読み手の**感情の流れ設計**（否定→共感→解放 の3ステップ等）
- Insight Cast のターゲット層への**言葉選びの解像度**

コピーを書く時は、フレームワーク（copywriting-lp）と思考法（このファイル）の両方を使う。

## 使う場面

- LPのセクション単位のコピーを書く時
- 「案は出たが何かが弱い」と感じる時
- コピーを提出する前のself-checkに
- レビュアー がコピーの論理・感情・リズムを評価する時
- 前後のセクションと呼応しているかを確認したい時

---

## マーケターが守る最低限のルール

コピーを出す前に必ず以下を自問する。

1. このコピーは A否定→B提示 の構造になっているか、それとも AとBを並べているだけか
2. openingの構文と closingの構文は対応しているか
3. 読み手の感情は「否定された → 共感された → 救われた」の順で動くか
4. 40〜60代の非IT・中小企業主に、読んだ瞬間に意味が届くか
5. 論証がくどくなっていないか（A でも B でもなく C は、1回まで）

自問して「分からない」と感じたら `lp-copy-principles.md` を読んでから書き直す。

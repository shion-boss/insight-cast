# Model Cost Review

## 目的
品質だけ、価格だけでモデルを決めず、両方の観点で比較する。

## 使う場面

- 新しいモデルを提案する時
- 既存モデルを切り替える時
- 複数段の AI 構成に変更する時

## 比較観点

1. 必要な知能レベル
2. 応答速度
3. 想定リクエスト量
4. 1リクエストあたりコスト
5. 月次固定費モードへの影響
6. 安価な代替案の有無

## 出力フォーマット

```md
## Model Review

### 対象機能
- ...

### 品質要件
- ...

### 候補
| model | quality fit | speed | cost | note |
|---|---|---|---|---|
| ... | ... | ... | ... | ... |

### 推奨
- ...

### リスク
- ...

### 人間に確認したいこと
- ...
```

## 注意

- 「高性能だから」で決めない
- 月次コストの見立てを書かずに承認へ進めない

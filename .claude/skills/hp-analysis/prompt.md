# HP分析プロンプト

## 現行プロンプト（`analyzeHp` 関数内）

```
以下はあるビジネスのホームページの内容です。日本語で分析してください。

## ホームページ内容
${markdown.slice(0, 8000)}

## 出力形式（JSONのみ返してください）
{
  "current_content": ["現在伝えていること（3〜4項目）"],
  "strengths": ["強みとして見えるもの（2〜3項目）"],
  "gaps": ["伝えきれていないこと・足りないもの（2〜3項目）"],
  "suggested_themes": ["インタビューで深めたいテーマ（3〜5項目）"]
}
```

---

## 各項目の設計意図

### `current_content`（現在伝えていること）

3〜4項目という指定の意図: 多すぎると後段処理の負荷が上がる。少なすぎると「読んだ」内容の網羅性が低い。3〜4項目は「主要な情報を整理した」レベルとして適切。

この項目はインタビューの前置きに使う。「御社のHPには○○と書いてありましたが〜」という会話の起点になる。

### `strengths`（強みとして見えるもの）

2〜3項目という指定の意図: 強みを多く出すよりも、「本当に見えるもの」に絞ることが重要。根拠のない強みリストを作るとインタビューでの検証が薄くなる。

HPを書いた人の意図と、外部から読んで伝わることの乖離を記録する観点。

### `gaps`（伝えきれていないこと）

「一般的なHPに書くべきこと」との比較ではなく、「このHP単体を読んで生じる疑問」として書く。
例: 「価格帯が全く記載されていない」「実績や事例が一切ない」「なぜこの事業者に依頼するべきかが分からない」

2〜3項目に絞ることで、インタビューで深めるべき優先順位を作る。

### `suggested_themes`（インタビューテーマ）

3〜5項目という指定の意図: インタビュー1回で扱えるテーマには限りがある。多すぎると散漫になる。3〜5項目のうち、キャストが自然な会話の流れで1〜2テーマを深く掘り下げる。

このテーマは将来的にキャストのシステムプロンプトに渡すことを想定している。

---

## 現行実装の限界と改善案

### 限界1: テキスト8000文字上限

`markdown.slice(0, 8000)` で切り捨てている。トップページしか読んでいないケースや、コンテンツが多いHPでは後半が読まれない。

**改善案**: Firecrawlのクロール機能を使い、主要ページ（会社概要・サービスページ・よくある質問など）を複数ページ取得して統合する。

### 限界2: システムプロンプトがない

現行はシステムプロンプトなし、ユーザーメッセージのみ。
役割設定がないため、出力の一貫性が低下するリスクがある。

**改善案**: システムプロンプトに「あなたはHPを読んで発信不足を見つける専門家です」という役割設定を追加する。

### 限界3: suggested_themes がキャストに渡っていない

現状、HP分析の結果はレポート画面に表示されるが、インタビュー開始時にキャストのプロンプトへは渡されていない。

**改善案**: インタビュー開始時に `hp_audit` と `competitor_analyses` の結果をシステムプロンプトに差し込む。これにより「HPに書かれていない〇〇について聞かせてください」という具体的な問いが可能になる。

## 改善版プロンプト（試案）

現行より精度が上がることが期待される改善版。採用前に人間のレビューが必要。

```
システムプロンプト:
あなたはホームページを読んで、発信できていない価値を見つける専門家です。
ビジネスの成果につながる情報の不足を見つけることが仕事です。
「書かれていないこと」に着目してください。採点ではなく、可能性の発見が目的です。

ユーザーメッセージ:
以下は事業者のホームページの内容です。日本語で分析してください。

## ホームページ内容
${markdown.slice(0, 8000)}

## 分析の観点
- このHPを初めて見た人は、この事業者に依頼したいと思うか
- 他の同業者との違いが伝わるか
- 具体的なエピソードや数字・実績があるか
- どんな人に向けたサービスか分かるか

## 出力形式（JSONのみ返してください）
{
  "current_content": ["現在伝えていること（3〜4項目）"],
  "strengths": ["強みとして見えるもの（2〜3項目）"],
  "gaps": ["伝えきれていないこと・不足していること（2〜3項目）"],
  "suggested_themes": ["次のインタビューで掘り下げると価値が高いテーマ（3〜5項目）"]
}
```

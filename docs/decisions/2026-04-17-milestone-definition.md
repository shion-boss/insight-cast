# 決定: Phase 2 中間マイルストーンの定義

## 日付

2026-04-17

## 背景

CLAUDE.md「フェーズと現在地」に記載の通り、Phase 2 のゴールは「Insight CastのHPに問い合わせが来る状態」とされているが、具体的な数値定義が未確定だった。Phase 0 の最重要宿題として、growth が定義案を策定した。

ドッグフーディング戦略の骨子（CLAUDE.md）より:
- MVPが動く状態になったら Insight Cast 自身のHPを作る
- そのHPのブログ更新を Insight Cast 自身を使って進める
- そのHPで問い合わせが来る状態を作る（= 中間マイルストーン）
- そこまで来て初めて、サブスク5件を本格的に取りにいく

## 検討した選択肢

### 選択肢A: 問い合わせ1件で達成とする
- メリット: 達成ハードルが低く、Phase 3 への移行が早い
- デメリット: 1件は偶然の可能性が消えない。「再現性の証明」にならない

### 選択肢B: 問い合わせ月2件 + 月500セッション + ブログ10本（AND条件）
- メリット: 「流入量」「コンテンツ蓄積」「問い合わせ」の3軸を同時に証明できる。どこが詰まっているかを特定しやすい
- デメリット: 達成のハードルが複合的になる。ただしこの複合性がドッグフーディングの実証として価値がある

### 選択肢C: 問い合わせ3件以上のみ
- メリット: 指標がシンプル
- デメリット: 流入量の担保がないため「たまたま3件来た」と区別できない

## 決定内容

**選択肢Bを採用する（人間の承認待ち）**

### 定量目標（AND条件）

> 「知り合い紹介を除いた問い合わせが月2件以上 かつ 月500セッション以上 かつ ブログ累計10本以上」を1ヶ月間維持できた状態

| 指標 | 目標値 | 根拠 |
|---|---|---|
| 月間オーガニック問い合わせ件数（紹介除く） | 2件以上 | 再現性が証明できる最小数。CVR 0.4% × 500セッションと整合 |
| 月間セッション数 | 500以上 | 問い合わせ率の分母。流入不足とCVR不足を切り分けるための基準値 |
| ブログ公開本数（累計） | 10本以上 | ドッグフーディングの実証コンテンツとして最低限必要な量 |

### 計測方法

- **セッション数・流入元**: Google Analytics 4（GA4）を自社HPに設置。流入元別（Organic / Direct / Referral）で追跡
- **問い合わせ件数**: 問い合わせフォームのサンクスページ到達イベントをGA4で計測。スプレッドシートに日付・流入元・業種を手動記録
- **紹介経由の除外**: 問い合わせフォームに「どこで知りましたか？」の任意記入欄を設置。「知人から聞いた」と明記されたものはカウント対象外
- **ブログ本数**: ページ一覧を目視カウント（Phase 3 以降に自動化を検討）

### 達成期限の目安

- Phase 1 完了まで: 2〜3ヶ月
- Phase 2 開始から中間マイルストーン達成まで: 3〜5ヶ月
- **合計: プロジェクト開始から5〜8ヶ月以内**

内訳根拠:
- ブログ月10本ペース（2026-04-22 運営より変更。旧: 月2本）
- 月10本ペースなら累計10本は1ヶ月で達成。SEO・AI検索での発見には最低3ヶ月の蓄積期間が必要

### 行き詰まりの判断基準

以下のいずれかが **2ヶ月連続**で続いた場合、方針転換を検討する:

1. ブログを月2本以上公開しているのに月間セッションが200を下回り続けている（流入不足）
2. 月間セッション300以上あるのに問い合わせが0件のまま（LP/CTAに問題がある可能性）
3. 代表者取材でブログのネタが枯渇し、月1本も公開できない（コンテンツ枯渇）
4. 自社HPは動いているが中小企業からの問い合わせが来ない（ターゲットレイヤーのズレの顕在化）

### 方針転換時の選択肢（ストック）

| 行き詰まりのタイプ | 転換オプション |
|---|---|
| 流入不足 | SNS発信（Facebook / Instagram）を強化し、HP誘導を増やす。YouTube「取材してみた」動画の先行実験 |
| LP/CTAに問題 | 問い合わせフォーム到達前のページを見直す。「まず1回無料で体験」などフック強化 |
| コンテンツ枯渇 | AIデザイナー にテーマ設計を再依頼。思想・哲学系テーマ（Insight Castを作った理由等）に軸を移す |
| ターゲットレイヤーのズレ | 地域中小企業への個別アプローチルートの先行実験（展示会・商工会・地域SNSグループ） |
| 全体的な手応えなし | growth と lead で戦略をゼロから再検討。「まず紹介で1件取ってから型を作る」路線への切り替えを人間に提案 |

## 理由

- AND条件にすることで「なぜ問い合わせが来ないか」の診断を指標ごとに分解できる
- 500セッションは「流入施策が機能しているか」を測る最小の数値として妥当
- ブログ10本はドッグフーディングの実証として顧客に見せられるコンテンツ量の基準
- 問い合わせ2件は「再現性の証明」として必要な最小数
- 方針転換の判断基準を事前にストックすることで、行き詰まった時の意思決定コストを下げる

## 影響を受ける場所

- `CLAUDE.md` の「中間マイルストーン（Phase 2 のゴール）」セクション
  - 承認後、lead が具体的な数値を CLAUDE.md に反映することを推奨

## 見直し条件

- Phase 2 開始から3ヶ月時点で進捗をレビューし、数値目標の妥当性を再評価する
- 市場環境や競合状況に大きな変化があった場合
- 代表者から追加の方針変更指示があった場合

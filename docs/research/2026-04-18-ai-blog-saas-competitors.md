# AIブログツール / SaaS 競合メモ

調査日: 2026-04-18

## ざっくり結論

- 現在の主要プレイヤーは、`SEO / GEO 最適化`、`ブランドボイス管理`、`量産ワークフロー化` を強く打ち出している。
- Insight Cast が勝ちやすいのは、その前段にある `何を伝えるべきかの発見`、`事業者本人の一次情報の引き出し`、`ホームページ改善の芯づくり`。
- つまり、真正面から「記事生成速度」で戦うより、`書く前の発見と整理` を明確にしたほうがポジションが立つ。

## 主要な競合の見え方

### Jasper

- 公式サイトは、ブログ執筆、SEO、マーケ運用の効率化を前面に出している。
- 価値の中心は `write / edit / optimize / collaborate` にある。
- 参考:
  - https://www.jasper.ai/use-cases/blog-writing
  - https://www.jasper.ai/use-cases/seo
  - https://www.jasper.ai/

### Writesonic

- 公式訴求は、`ranking content`、`real-time research`、`fact-checking`、`competitor analysis` が中心。
- 最近は AI Search / visibility tracking の色も強い。
- 参考:
  - https://writesonic.com/lp/ai-article-writer
  - https://writesonic.com/
  - https://writesonic.com/lp/ai-content-writer

### Frase

- `Rank on Google. Get cited by AI.` のように SEO + GEO を一体で打ち出している。
- 研究、執筆、最適化、可視性追跡までを一つの運用基盤として見せている。
- 参考:
  - https://www.frase.io/
  - https://www.frase.io/features/agent
  - https://www.frase.io/ai-content-brief-generator

### Anyword

- ブランドボイスと予測パフォーマンスが中心。
- 「どのコピーが成果につながるか」を事前に見せる、広告 / GTM チーム向けの色が強い。
- 参考:
  - https://www.anyword.com/
  - https://www.anyword.com/data-driven-editor
  - https://www.anyword.com/performance-marketers

### Semrush / HubSpot

- Semrush は Content Toolkit で、調査、ブリーフ、生成、最適化、再利用、公開までを一気通貫で提供。
- HubSpot Breeze は、既存 CRM やマーケ運用とつながるコンテンツ生成エージェントとして訴求。
- どちらも「すでに運用するチーム」が主対象に見える。
- 参考:
  - https://www.semrush.com/kb/812-content-toolkit/
  - https://www.semrush.com/kb/1535-getting-started-with-content-toolkit
  - https://www.hubspot.com/products/content/content-ai-agent

## Insight Cast で強めるべき訴求

- `AIブログツールではなく、書く前の取材サービス`
- `ホームページと競合を先に読んでから聞く`
- `社長や担当者が答えるだけで、強みの芯が見えてくる`
- `記事本文だけでなく、FAQ、比較軸、サービス説明にも効く`
- `SEO / GEO / ブランド運用ツールとは併用できる`

## このラウンドでサイトに反映した方向

- トップページに `AIブログツールとの役割の違い` を明示
- サービスページに `置き換える関係ではない` 比較を追加
- 料金ページに `どこまで伴走が必要か` と `併用の考え方` を追加
- FAQ に `一般的なAIブログツールとの違い` と `併用可否` を追加
- CTA 文言を、より具体的な `無料で取材を始める` に寄せた
